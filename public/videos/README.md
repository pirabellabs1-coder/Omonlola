# Vidéos

Déposez ici votre showreel sous le nom **`showreel.mp4`** (référencé dans `components/sections/VideoSection.tsx`).

Pour changer le poster (image d'aperçu) ou l'URL, modifiez les constantes en haut du fichier :

```ts
const VIDEO_SRC = "/videos/showreel.mp4";
const POSTER_SRC = "https://...";
```
